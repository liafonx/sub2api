Design Details

  • Trigger: Click "Service Accounts" card → accounts popup. Click "New Users Today" card → users popup.
  • Entity cards: Name + platform tag, 3 bold metric numbers with color coding
  • Peak vs Max: Concurrency shows "peak / configured max" for context. Sessions and RPM show just the peak number.
  • Reset All: Red-tinted button in footer. Triggers ConfirmDialog with danger styling.
  • After reset: All values show 0, toast notification confirms. Tracking restarts immediately.
  • Scrollable: Max-height with overflow-y for deployments with more accounts.
  • Mobile: Metrics wrap to 2 columns on narrow screens. BaseDialog handles responsive width.
